# Gry Dropzone z Commodore 64.

# Dropzone from Commodore 64
## This isn't game port one to one.

Controls:
* Enter - stop/pause
* Up/down/left/right - dropship control
* Q - invisibility and immortality
* Crtl - main gun
* Space - area attack
